# Data 目录说明

以下为按内容规整后的文件名（原长文件名已重命名）：

| 新文件名 | 内容说明 |
|----------|----------|
| `行情_上证50.xlsx` | 上证50指数行情（交易日期、收盘价、涨跌幅等） |
| `行情_上证1000.xlsx` | 上证1000指数行情 |
| `行情_沪深300.xlsx` | 沪深300指数行情 |
| `因子_国家队_pair_zscore.xlsx` | 国家队因子（多 sheet：pct_chg, netinflow 等） |
| `因子_国家队_增强版_pair_zscore.xlsx` | 增强版国家队因子（多 sheet） |
| `数据_波动率与股债指数.xlsx` | 波动率、股债风险指数等（多 sheet） |

说明：以 `~$` 开头的文件为 Excel 临时锁文件，可忽略。
